# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Casefile, please report it privately by opening a security advisory at:

https://github.com/x4cc3/casefile/security/advisories/new

Do not report security vulnerabilities through public GitHub issues.

We will acknowledge receipt within 48 hours and provide an estimated timeline for a fix. Once the vulnerability is resolved, we will publish a security advisory and credit the reporter.

## Scope

This security policy covers the Casefile plugin for pi and Codex, including its MCP server, skills, and associated tooling.

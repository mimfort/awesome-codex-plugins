# Link Outreach Acquisition Examples

Examples for subject lines, responses, and tracking.

## Subject Line Patterns

| Outreach Type | Pattern |
|---------------|---------|
| Resource page | `Suggestion for [resource page]: [resource]` |
| Broken link | `Broken link on [site]: right person?` |
| Attribution | `Attribution request: [article title]` |
| Content tip | `Tip for [site]: [content title]` |
| Roundup | `Suggestion for [roundup name]` |
| Digital PR | `Story idea: [data/story title]` |
| Guest contribution | `Article idea for [site]: [topic]` |

## Response Handling

| Response | Action |
|----------|--------|
| Ignore | Wait, send one custom follow-up with original below, then suppress if still silent |
| Accept | Thank, record placement, verify link, ask what future resources help |
| Decline | Thank, ask whether future relevant pings are welcome, record preference |
| Counter | Evaluate request; paid-link counters require risk review and should not be treated as earned outreach |

## Tracking Fields

- Prospect host and target page.
- Contact name, role, email, social handle.
- Site type and segment.
- Asset pitched and destination URL.
- Subject line and opening line.
- Request type.
- Contact and follow-up dates.
- Response category and notes.
- Placement URL, linked URL, anchor/context, and date placed.
- Relationship history, opt-out status, and future interests.

## Weak To Strong Email Framing

| Weak | Strong |
|------|--------|
| "Great post, please link to us." | "Your resource page includes tools for first-time buyers. Our checklist fills the financing-prep gap and may help that audience." |
| "Can you add our link?" | "Would you consider including this guide as a resource for readers comparing options?" |
| "Following up again..." | "Sharing one extra detail in case useful: the guide now includes the downloadable worksheet you mentioned in your article." |

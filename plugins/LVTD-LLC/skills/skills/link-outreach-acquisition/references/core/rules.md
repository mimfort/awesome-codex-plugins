# Link Outreach Acquisition Rules

Use these rules when drafting, reviewing, or managing link outreach.

## Core Rules

### 1. Match Asset To Prospect

Do not outreach until the asset and prospect fit are clear.

- Name the exact asset or URL.
- Name the target page or editorial context.
- State why the publisher's audience benefits.

### 2. Use Honest Specificity

Subject lines and emails should set accurate expectations.

- Name the resource page, article, site, asset, or request type when useful.
- Avoid fake "Re:", fake follow-ups, fake friendship, or fake prior conversations.
- Use words like share, cite, include, update, credit, or suggest when they better match the ask.

### 3. Personalize With Context

Personalization must connect to the request.

- Use a relevant article, author beat, resource gap, broken URL, unlinked mention, or prior share.
- Do not rely on vague praise.
- Customize more for high-value prospects.

### 4. Make The Request Easy

Reduce work for the recipient.

- Include exact URLs.
- Identify the target page on their site.
- Keep the email short.
- Offer a low-friction next step.

### 5. Track And Respect Responses

Maintain contact history and suppression data.

- Track subject line, opening line, asset, response, placement, opt-out, and future interest.
- Thank accepts and declines.
- Stop contacting opt-outs.

## Email Review Rubric

Score 0-2. A sendable email should have no zeroes on relevance, honesty, audience benefit, or compliance.

| Criterion | Strong Signal |
|-----------|---------------|
| Prospect fit | Clear match to site, author, page, or beat |
| Personalization | Specific context tied to request |
| Audience benefit | Explicit reader/editor value |
| Subject line | Honest and specific |
| Request wording | Natural action for the publisher |
| Specificity | Exact target page and destination URL |
| Compliance | Truthful identity, non-deceptive subject, opt-out path when needed |

## Red Flags

- Generic "great post" personalization.
- Wrong name, site, or article.
- Vague "link to us" request.
- Deceptive subject line.
- Large batch with no segmentation.
- Follow-up ignores a decline or opt-out.

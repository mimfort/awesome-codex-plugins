# Link Outreach Acquisition

Use this skill to plan, draft, review, and manage outreach that asks publishers, bloggers, journalists, resource curators, or site owners to mention, share, cite, include, credit, or update a linkable asset.

This skill is derived from Garrett French and Eric Ward's *Ultimate Guide to Link Building* and uses transformed guidance with durable book-topic references. Do not copy book prose into user outputs.

## Quick Start

1. Use the Contents table below to choose the smallest useful reference set.
2. Use `workflows/run-link-outreach.md` for a full campaign workflow.
3. Confirm the prospect has been qualified and the asset fits the audience.
4. Write specific, honest, low-friction outreach.
5. Track every contact, response, placement, opt-out, and relationship note.

## Default Output

When asked for outreach help, return:

1. **Campaign context** - outreach type, target asset, prospect segment, and goal.
2. **Asset-to-audience fit** - why this prospect's audience benefits.
3. **Personalization notes** - specific context tied to the ask.
4. **Subject lines** - honest options by request type.
5. **Email draft or review** - concise pitch with exact URLs and next step.
6. **Tracking and response plan** - fields, follow-up timing, response categories, and compliance risks.

## Core Posture

- Pitch value to the publisher's audience, not value to your SEO campaign.
- Personalization must be contextual and tied to the request.
- Be honest about identity, intent, and next step.
- Pilot scaled outreach before sending the whole list.
- Stop contacting people who opt out or clearly decline.

# Guest Post Placement Knowledge

Core concepts for editorial guest post campaigns.

## Key Concepts

### Guest Post Placement Campaign

A bylined-content process spanning prospecting, content design, approval, outreach, placement, and tracking.

### Citability

A link must make sense to the reader, editor, and quality reviewer. Strong citations support data, quotes, definitions, methods, resources, visuals, or claims.

### Sales Page Citability Gap

Commercial pages often need citable elements before guest posts can naturally reference them. Examples include proprietary data, expert quotes, definitions, pricing context, media, downloads, reviews, task advice, origin stories, or resource sections.

### Practitioner Action Cycle

Define the audience by work being done: tasks, skill level, setting, constraints, problems, emotional stakes, and the object affected.

### Audience Naming

Translate target keywords into audience or category names before prospecting. Think in directory-style categories, not only search terms.

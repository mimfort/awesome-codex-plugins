# Guest Post Placement Examples

Examples for citability, title planning, and tracking.

## Citable Elements For Commercial Pages

| Commercial Page Problem | Citable Addition |
|-------------------------|------------------|
| Product page has only sales copy | Add use-case guide, specs, expert advice, or comparison table |
| Pricing page cannot be cited naturally | Add pricing model explanation or cost calculator |
| Service page lacks authority | Add methodology, case evidence, FAQs, or downloadable checklist |
| Category page is thin | Add definitions, buying criteria, data, or examples |

## Pitch Title Examples

| Audience | Better Title Direction |
|----------|------------------------|
| New managers | Practical checklist or mistake-avoidance guide |
| Technical buyers | Evaluation criteria, integration questions, or benchmark interpretation |
| Local parents | Safety, scheduling, budget, or seasonal planning angle |
| Practitioners | Process improvement, tools, templates, or field lessons |

## Placement Quality Rubric

Score 0-3. A zero in audience fit, citation fit, or editorial legitimacy is a hard stop.

| Category | Strong Signal |
|----------|---------------|
| Audience fit | Clear named practitioner audience |
| Citation fit | Target page has a natural cited element |
| Editorial legitimacy | Real editor, guidelines, and standards |
| Reach | Social, comments, links, syndication, or known readers |
| Backlink trust | Natural links from recognizable sites |
| Outbound behavior | Useful balanced links |
| Expertise level | Fits available SME/data |
| Relationship value | Reusable publisher relationship |

## Tracking Fields

- Prospect URL, contact path, niche focus, guidelines URL, quality score.
- Title, audience benefit, citation target, content status.
- Contact dates, byline, placement URL, linked URL, and promotion actions.
- Do-not-contact and relationship notes.

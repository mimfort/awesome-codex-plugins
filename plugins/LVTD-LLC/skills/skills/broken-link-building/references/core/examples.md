# Broken Link Building Examples

Examples for tracking and outreach preparation.

## Fields To Track

| Field Group | Fields |
|-------------|--------|
| Prospect | campaign ID, source query, prospect domain, resource page URL, page type, topic fit, quality notes |
| Status | checked URL, source page URL, first status, recheck status, final status, dead type, redirect target, archive URL |
| Backlink | dead URL, dead topic, dead scope, referring domains, inbound links, authority metric, anchor text, referring page URL, link still present |
| Outreach | replacement URL, replacement fit, contact info, contact confidence, outreach status, live backlink URL, date acquired |
| Salvage | old URL, discovery source, current status, best live destination, redirect type, redirect verified |

## Replacement Fit Examples

| Fit | Meaning | Outreach Posture |
|-----|---------|------------------|
| Exact | Same topic and same user job | Strong suggestion |
| Close | Same topic, slightly different depth or format | Clear suggestion with caveat |
| Similar | Related resource but not equivalent | Use carefully or create better asset |
| Weak | Only loosely related | Do not outreach |
| None | No replacement | Save for future content idea |

## Outreach Note Elements

- Exact page containing the broken link.
- Dead URL or anchor text.
- Replacement URL.
- Short reason the replacement fits.
- Optional HTML snippet when useful.
- Affiliation disclosure.
- One clear request.

## Salvage Example

An old guide URL with many referring domains now returns 404. Map it to the updated guide if one exists. If no equivalent exists, create a replacement or redirect to the closest category page only when it still satisfies user intent.

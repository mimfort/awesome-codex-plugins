# Broken Link Building Rules

Use these rules when running broken link building or link equity salvage.

## Core Rules

### 1. Start With A Strong Replacement

Do not prospect until the promoted asset is useful and relevant.

- Record topic, audience, and replacement strength.
- Improve or create the asset if no credible replacement exists.
- Segment exact, close, similar, weak, and no-fit replacements.

### 2. Recheck Before Acting

Dead-link tools produce false positives.

- Recheck every dead or unresponsive URL.
- Use multiple checks for high-value dead URLs.
- Do not outreach until the referring page still shows the dead link.

### 3. Scrub Backlink Data

Backlink exports are not prospect lists.

- Deduplicate domains.
- Filter toward resource/link/reference/FAQ-style pages.
- Confirm the page loads and the dead link is present.
- Keep one selected outreach URL per domain unless there is a clear reason.

### 4. Be Honest In Outreach

Frame the note as maintenance help.

- Identify the exact page and dead link.
- Provide the replacement URL and fit rationale.
- Disclose affiliation when applicable.
- Do not pitch a weakly related page as an exact replacement.

### 5. Salvage To Closest Relevant Destination

For owned old URLs, map each dead URL to the closest live page.

- Do not redirect everything to the homepage.
- Verify redirect status and destination relevance.

## Red Flags

- Raw dead-link counts are reported before verification.
- Opportunity counts are promised before dedupe and contact review.
- Archive or anchor-text review is skipped.
- Same domain receives multiple avoidable requests.
- Contact discovery is trusted without manual selection.

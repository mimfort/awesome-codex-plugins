# Broken Link Building Knowledge

Core concepts for broken link building and link equity salvage.

## Key Concepts

### Broken Link Building

Find pages that link to dead, relevant resources and suggest the user's equivalent or better page as a fix.

### Resource Page Fit

Start with curated links, resources, references, FAQ lists, and similar pages where the user's asset would be useful even without the broken-link angle.

### Status Verification

Initial dead-link findings are only suspects. Recheck before metrics work, manual review, backlink export, or outreach.

### Replacement Fit

Exact one-to-one replacements usually work best. Similar-resource suggestions are weaker and should be described honestly.

### Link Equity Salvage

Reverse broken link building for the user's own site: find dead pages, folders, and subdomains with backlinks, then map them to the closest relevant live destination.

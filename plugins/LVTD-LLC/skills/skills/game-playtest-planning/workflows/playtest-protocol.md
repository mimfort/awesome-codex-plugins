# Playtest Protocol Workflow

## Procedure

1. Write 1-3 research questions.
2. State the decision each question will inform.
3. Select the participant profile.
4. Choose the method and session length.
5. Write the moderator script:
   - welcome
   - consent or recording note if applicable
   - no-help instruction
   - tasks or free-play prompt
   - debrief questions
6. Define observation tags and telemetry events.
7. Define the post-test synthesis table.

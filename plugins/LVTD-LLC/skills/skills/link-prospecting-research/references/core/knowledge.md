# Link Prospecting Research Knowledge

Core concepts for researching link opportunities before qualification or outreach.

## Key Concepts

### Link-Opportunity Type

A recurring page or publisher type that may naturally host a link. Examples include resource lists, guest posts, directories, profile listings, forums, Q&A pages, interviews, publications, roundups, events, sponsorship pages, and associations.

### Asset-Opportunity Fit

Start with the user's linkable assets and goals, then find pages that would naturally value those assets. A generic prospect list is weak if it does not explain why a specific asset fits.

### Market-Defining Keywords

Broad market terms used for research. Good market-defining keywords are short, common in the market, used by practitioners or publications, and broad enough to reveal the publishing ecosystem.

### Keyword Classes

| Class | Purpose |
|-------|---------|
| MDKW | Broad market-defining keywords |
| CDKW | Customer or practitioner labels |
| PCKW | Product or category terms |
| ITL | Industry thought leaders and experts |
| CCN | Competitor company names |
| GKW | Geographic terms |
| RVKW | Related or adjacent verticals |

### Query Anatomy

Strong prospecting queries combine:

- A research phrase.
- An advanced operator such as `intitle:`, `inurl:`, `site:`, or exclusions.
- A tactic-specific footprint such as "write for us," "resources," "guest post," "directory," or "roundup."

### Recalibration

If results are weak, adjust the research phrase, broaden or narrow the market term, add exclusions, test adjacent verticals, or change the opportunity type. Do not keep digging through low-quality results.

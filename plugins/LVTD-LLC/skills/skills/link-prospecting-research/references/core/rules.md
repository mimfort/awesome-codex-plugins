# Link Prospecting Research Rules

Use these rules when researching possible link prospects.

## Core Rules

### 1. Start From Asset Fit

Map assets to opportunity types before generating query batches.

- Identify what the user can credibly offer.
- Name the page type or publisher type that would value it.
- Reject generic lists that cannot explain fit.

### 2. Build A Keyword Universe

Use more than transactional keywords.

- Include market, customer, product, expert, competitor, geo, and adjacent vertical terms.
- Use broad market words to discover the publishing ecosystem.
- Use narrower terms to segment and reduce noise.

### 3. Segment Queries

Create many precise query segments instead of digging deep into weak results.

- Combine research phrase, operator, and footprint.
- Save useful footprints from strong prospects.
- Compare opportunity types with consistent result windows.

### 4. Preserve Source Evidence

Record where prospects came from.

- Keep source query, source list, competitor page, backlink export, or autocomplete seed.
- Deduplicate across all sources.
- Keep candidates separate from qualified prospects.

### 5. Keep Research Separate From Outreach

Prospecting produces candidates and evidence. Qualification and outreach are later steps.

## Red Flags

- Raw search-result counts are treated as opportunity size.
- Competitor backlinks are treated as validated prospects.
- Forum or community links are suggested without participation value.
- Paid, sponsored, editorial, profile, nofollow, and resource opportunities are blended together.
- Autocomplete or list-scrape noise is not filtered.

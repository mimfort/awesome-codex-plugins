# Link Prospecting Research Examples

Examples for building query libraries and opportunity maps.

## Query Pattern Templates

Use placeholders as variables.

| Goal | Query Patterns |
|------|----------------|
| Blogs and publications | `[MDKW] blog`, `[MDKW] "blog list"`, `"top [MDKW] blogs"`, `[MDKW] "trade publication"` |
| Niche directories | `"[MDKW] directory"`, `"[MDKW] websites"`, `[MDKW] "suggest * URL"` |
| Interviews and experts | `[MDKW] intitle:interview`, `[ITL] interview`, `intitle:"[ITL]"`, `[ITL] forum` |
| Communities and associations | `[MDKW] community`, `intitle:[MDKW] forum`, `[MDKW] association` |
| Resource curators | `[MDKW] roundup`, `[MDKW] resources list`, `"useful [MDKW] links" library` |
| Guest placement | `[RESEARCH_PHRASE] "guest post"`, `[RESEARCH_PHRASE] "write for us"`, `[RESEARCH_PHRASE] "about the author"` |
| Competitor signals | `[CCN] review`, `[CCN] forum`, `[CCN] interview`, `[CCN] "guest article"` |
| Local or adjacent verticals | `[GKW] events`, `[GKW] directory`, `[RVKW] "guest post"`, `"[RVKW] resources"` |
| List scraping | `list of [TARGET_TYPE]s`, `top [N] [TOPIC] blogs`, `[TOPIC] directories`, `[TOPIC] resources` |
| Autocomplete-derived blogs | `[SUGGESTED_INFO_NEED] inurl:blog`, `[SUGGESTED_INFO_NEED] blog` |

## Opportunity Type Example

For a kayaking retailer with guides, experts, and local events:

- Resource pages: guides to kayaking spots.
- Blogs: safety, travel, and event stories.
- Forums: expert participation, not link dropping.
- Local sites: event listings and regional directories.
- Adjacent verticals: hiking, camping, and outdoor recreation publishers.

## Weak To Strong Research

| Weak | Strong |
|------|--------|
| Search `[industry] blog` once | Build query batches by asset, audience, footprint, and adjacent vertical |
| Export competitor backlinks and email everyone | Categorize why each link exists, then qualify fit |
| Scrape one list | Combine multiple source lists, dedupe, and preserve source evidence |
| Use only product keywords | Add practitioner labels, thought leaders, competitors, geo, and related verticals |

# Plan Linkable Asset Workflow

Use this workflow to plan or audit assets before link prospecting and outreach.

## When to Use

- The user needs content or assets that can earn links.
- A campaign has prospects but no strong asset.
- Existing assets may be linkable but need evaluation.
- The user wants competitor-inspired asset ideas.

## Prerequisites

- Website or business description.
- Target market or topic.
- Current assets if available.
- Business goal and target audience.

**Reference**: `references/core/knowledge.md`, `references/core/rules.md`

## Workflow Steps

### Step 1: Define Linker Audiences

**Goal**: Name who might link and why.

- [ ] List likely publishers, curators, communities, experts, journalists, partners, and local groups.
- [ ] State the audience each linker serves.
- [ ] Identify the tasks, pains, or decisions that audience needs help with.

### Step 2: Inventory Existing Assets

**Goal**: Find underused linkable material.

- [ ] Review guides, tools, data, PDFs, videos, events, jobs, experts, partner pages, and product education.
- [ ] Note pages that already earned links.
- [ ] Mark assets that need refresh, packaging, design, internal links, or promotion.

### Step 3: Study Market Evidence

**Goal**: Learn what the market already links to.

- [ ] Review competitor and publisher most-linked pages.
- [ ] Record format, topic, audience, structure, and likely reason for links.
- [ ] Mark ideas as proven, saturated, weakly served, or missing.

### Step 4: Map Pains To Formats

**Goal**: Convert real needs into asset ideas.

- [ ] Group recurring pains and search patterns.
- [ ] Choose formats: guide, tool, template, data study, expert survey, resource hub, checklist, webinar, giveaway, or local resource.
- [ ] State the value add over existing resources.

### Step 5: Prioritize Concepts

**Goal**: Pick assets worth producing.

- [ ] Score each idea for link potential, effort, differentiation, audience fit, and business fit.
- [ ] Identify required contributors and blockers.
- [ ] Choose the first validation or production step.

## Common Mistakes

| Mistake | Why It's Bad | Do Instead |
|---------|--------------|------------|
| Starting with brand message | Publishers need audience utility | Start with linker audience pains |
| Skipping asset audit | Misses quick wins | Review current and historical assets |
| Copying competitors | Creates undifferentiated content | Improve angle, depth, utility, or format |
| Thin resource lists | Not worth citing | Add criteria, summaries, categories, or scoring |

## Exit Criteria

Task is complete when:

- [ ] The plan names linker audiences and their needs.
- [ ] Existing assets were reviewed.
- [ ] Market evidence supports the recommended concepts.
- [ ] Each concept includes format, value add, resources, prospects, and next action.

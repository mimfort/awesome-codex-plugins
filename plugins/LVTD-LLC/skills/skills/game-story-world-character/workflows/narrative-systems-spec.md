# Narrative Systems Spec Workflow

## Procedure

1. Name the gameplay purpose of story in this project.
2. Write the core premise in one paragraph.
3. Map major beats to player actions and game state.
4. Define world rules that affect mechanics, levels, UI, audio, art, or progression.
5. Define characters by gameplay function first.
6. Specify content data structures and triggers.
7. Add tests for branching, repeated triggers, skipped content, and invalid state.

## Story-Gameplay Map

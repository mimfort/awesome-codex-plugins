# Manuscript Engagement Analytics Checklist

Use this checklist when auditing a manuscript for engagement and value pacing.

## Setup

- [ ] Target reader is known.
- [ ] Book promise is known.
- [ ] First meaningful payoff is defined.
- [ ] Manuscript or outline has headings.
- [ ] Word counts are available or generated.
- [ ] Beta comment/dropoff data is available if relevant.

## Structure Map

- [ ] Headings are listed with line numbers.
- [ ] Section word counts are calculated.
- [ ] Cumulative word counts are calculated.
- [ ] Long sections are flagged.
- [ ] Vague or topic-only headings are flagged.

## Value Events

- [ ] First payoff distance is estimated.
- [ ] Practical tools, examples, checklists, labs, and decisions are marked.
- [ ] Runs of setup without payoff are flagged.
- [ ] Hidden value is promoted where found.
- [ ] Sections outside reader scope are marked.

## Beta Engagement

- [ ] Comment clusters are mapped.
- [ ] Comment dropoffs are mapped.
- [ ] Abandonment points are compared across readers.
- [ ] Follow-up questions are listed for ambiguous dropoff.
- [ ] Nearby sections are inspected for boredom and confusion.

## Recommendations

- [ ] Highest-risk sections are ordered.
- [ ] Structural fixes come before prose polish.
- [ ] Suggested cuts preserve useful material when it can become companion content.
- [ ] Revised headings or sequence are proposed where useful.
- [ ] Follow-up check after revision is defined.

## Red Flags

- More than one chapter passes before concrete value.
- Several long sections have topic labels instead of takeaways.
- Readers stop commenting near the same point.
- The author explains why readers should care instead of giving them value.
- Metrics are used as verdicts instead of inspection signals.

## Quality Rubric

| Dimension | Strong | Weak |
|-----------|--------|------|
| Evidence | Word counts plus reader value | Vague impression |
| Diagnosis | Names payoff delay or dropoff cause | Says "make it shorter" |
| Recommendation | Move, cut, split, rename, add value | General writing advice |
| Beta use | Interprets behavior cautiously | Assumes all silence is boredom |

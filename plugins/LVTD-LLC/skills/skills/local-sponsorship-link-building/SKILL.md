# Local Sponsorship Link Building

Use this skill to plan local sponsorship campaigns that create local visibility, community relationships, and legitimate links without treating sponsorships as simple link purchases.

This skill is derived from Garrett French and Eric Ward's *Ultimate Guide to Link Building* and uses transformed guidance with durable book-topic references. Do not copy book prose into user outputs.

## Quick Start

1. Use the Contents table below to choose the smallest useful reference set.
2. Use `workflows/plan-local-sponsorship-campaign.md` for a full campaign.
3. Clarify geography, goals, audience, budget, restrictions, and required benefits before prospecting.
4. Evaluate sponsorships by local relevance, audience fit, fulfillment reliability, cost, and relationship upside.
5. Return prospecting plan, scored shortlist, fulfillment checklist, measurement plan, and relationship/blog add-ons.

## Default Output

When asked for local sponsorship link building, return:

1. **Campaign brief** - geography, goals, audience, budget, constraints, timeline, and required benefits.
2. **Prospecting plan** - queries, categories, seasonal angles, and inclusion/exclusion criteria.
3. **Outreach plan** - email/call sequence, tracking fields, response window, and personalization notes.
4. **Evaluation matrix** - scored shortlist using the sponsorship rubric.
5. **Recommended sponsorships** - ranked opportunities with rationale, risks, and required assets.
6. **Fulfillment checklist** - confirmations, payment steps, asset packet, deadlines, and benefit checks.
7. **Measurement plan** - direct attribution, SEO tracking, local brand/reach metrics, and reporting cadence.
8. **Relationship add-ons** - existing relationship and local blog ideas when relevant.

## Core Posture

- Sponsorships are hyper-local marketing, not paid links.
- Links are one benefit among reach, trust, relationships, local PR, and community presence.
- Confirm brand restrictions, payment logistics, and fulfillment details before outreach scales.
- Measure benefits beyond immediate sales.

# Local Sponsorship Link Building Rules

Use these rules when planning or reviewing local sponsorship campaigns.

## Core Rules

### 1. Define Parameters Before Prospecting

Capture geography, audience, budget, target count, deadline, required benefits, approvals, restrictions, and payment path.

### 2. Treat Sponsorship As Community Marketing

Do not frame the campaign as buying links.

- Evaluate reach, mission fit, relationship upside, and local relevance.
- Links should be a legitimate sponsorship benefit, not the only rationale.

### 3. Confirm Brand And Payment Constraints

Ask about restrictions before outreach: religious or political organizations, controversial categories, competitor adjacency, required nonprofit status, ethics concerns, and focus areas.

### 4. Score More Than Link Metrics

Evaluate goal fit, local relevance, audience fit, link value, reach, mission fit, timing, fulfillment reliability, cost efficiency, and relationship upside.

### 5. Fulfill Cleanly

Provide assets and confirm timing.

- Logo variants, local URL, UTM links, company description, coupon code if available, social/newsletter copy, print files, and deadlines.
- Verify live link, URL, logo, posts, newsletters, coupon code, and benefit duration.

## Red Flags

- Sponsorship is chosen by DA or price alone.
- Organizer cannot confirm benefits, timing, or asset needs.
- Brand restrictions are unknown.
- The link target is a generic homepage when a useful local page exists.
- User expects immediate last-touch sales attribution.
- Follow-up cadence risks damaging local relationships.

# Responsibility Audit Workflow

## Procedure

1. State the game's purpose and target audience.
2. List systems that apply pressure or shape behavior.
3. Identify vulnerable contexts and misuse scenarios.
4. Rate each risk by severity and likelihood.
5. Recommend mitigations with owners.
6. Define ship gates and telemetry.

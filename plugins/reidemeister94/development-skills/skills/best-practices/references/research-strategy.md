# Domain Authority Anchors

Non-obvious starting-point sources per domain — the ones not already top-of-mind. The derivable top-tier (official docs/RFCs/seminal papers, NEJM/Lancet/JAMA/Cochrane-class journals, central-bank/APA research) is assumed; reach past it to these:

- **Software:** Pragmatic Engineer, Architecture Notes, ThoughtWorks Technology Radar, ByteByteGo, ACM Queue; top-company engineering blogs (Netflix, Stripe, Cloudflare, Meta).
- **Health / Medicine:** Cochrane systematic reviews, UpToDate, Mayo/Cleveland/Johns Hopkins.
- **Fitness / Exercise Science:** Schoenfeld, Helms, Israetel, Nuckols; Stronger By Science, Renaissance Periodization (research-backed).
- **Nutrition:** Examine.com systematic reviews (the anchor), EFSA scientific opinions, Precision Nutrition.
- **Finance:** Vanguard / AQR / DFA research, CFA Institute, Bogleheads; foundational authors (Bogle, Malkiel, Damodaran).
- **Psychology:** foundational researchers (Kahneman, Tversky, Bandura, Dweck), NICE guidelines.
- **Cooking / Food Science:** Harold McGee (*On Food and Cooking*), Modernist Cuisine, ATK / Cook's Illustrated blind testing, Kenji López-Alt; ChefSteps, Serious Eats test kitchen.

## Quality gate for non-authoritative sources
GitHub repos / community sources count only at **≥1000 stars AND active within the last 6 months**. Tag claims `[strong]` / `[contested]` / `[convention]` per SKILL.md Step 4; a `[convention]` (widely-followed practice, not evidence-backed) must be labeled as such, never as `[strong]`.
